# PDCAリフレクション スキル

`criteria/evaluator-criteria.md` を読み込み、過去の学習を現在の作業に反映するスキルです。
Planner・Evaluator の両エージェントが共通で使用します。

## 手順

1. `criteria/evaluator-criteria.md` を読み込む
2. 「蓄積された基準」セクションを確認する
   - **基準が空の場合**: PDCA改善なし。「初回実施」として扱う
   - **基準が蓄積済みの場合**: 該当する基準を現在の作業に適用する
3. 「禁止・注意事項」セクションを確認し、違反がないようにする

## Plannerでの適用

仕様書の「PDCAからの改善点（ルール4）」セクションに記載する:
- 蓄積基準から今回のタスクに関連するものを抜粋
- 前回と比べた改善方針を具体的に記述
- 蓄積なし → 「初回実施」と記載

## Evaluatorでの適用

採点の「蓄積された採点基準への適合（10点）」で使用する:
- 蓄積基準と成果物を照合し、適合度を採点
- 蓄積なし → この10点は仕様書品質の補足評価として使用する
